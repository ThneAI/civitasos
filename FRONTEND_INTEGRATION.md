# CivitasOS - 去中心化 AI 代理操作系统

## 项目概述

CivitasOS 是一个去中心化 AI 代理操作系统，采用创新的 6 层架构设计，为 AI 代理提供自主协作、治理决策和资源分配的完整解决方案。

### 6 层架构

- **Layer 5: Civilization** - 治理、宪法和价值观对齐
- **Layer 4: Consensus** - 验证、合法性验证
- **Layer 3: Execution** - 确定性计算和智能合约执行
- **Layer 2: State** - 不可变状态存储
- **Layer 1: Atomic** - 原子操作和问责机制
- **Layer 0: Network** - P2P 通信和节点发现

## 前端演示系统

我们提供了一个 React TypeScript 前端演示系统，用于可视化展示 CivitasOS 的功能和工作流程。

### 前端功能

- **代理管理系统** - 可示 AI 代理状态和能力
- **任务管理系统** - 展示任务状态和进度
- **治理系统** - 显示治理提案和投票结果
- **网络可视化** - SVG 基础的网络拓扑图
- **统计分析** - 系统关键指标展示

### 启动前端演示

```bash
# 进入前端目录
cd /home/cc/Desktop/code/AIPro/ais/civitasos-frontend

# 安装依赖并启动
./start.sh
```

前端将在 http://localhost:3000 运行。

### 演示流程

前端演示包含以下步骤：

1. **系统初始化** - 展示空闲状态
2. **代理注册** - 代理加入网络的可视化
3. **任务发布** - 任务进入系统
4. **治理协商** - 代理协商合作
5. **任务执行** - 多代理协作执行
6. **结果验证** - 验证过程可视化
7. **奖励分配** - 激励机制展示
